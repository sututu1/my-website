如何将代码上传到 GitHub?


2023/8/15


Git 使用方法,在终端上执行命令
1.用mkdir命令新建文件夹 (mkdir main);
2.用cd 切换到当前目录 (cd main);
3.用touch新建文件 (touch hello.js);
4.用 vi hello.js 添加文本内容 (vi hello.js),编辑console.log(“hello world”);
5.在 GitHub 新建新仓库 (main);
echo "# main" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/sututu1/main.git
git push -u origin main
按上述命令执行;
6.用git add +文件名 (git add hello.js);
7.用 git commit -m ‘更新文本内容/有意义的信息’ ;
8.用 git push命令拉取;
9.用 git pull 命令同步;
10.可以在 Git Hub网站查看文档;
