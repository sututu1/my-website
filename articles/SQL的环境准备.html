SQl的环境准备
1.首先安装docker环境,在浏览器下载docker;
2.安装mySQL的终端命令语句 docker pull mysql;
3.在终端上执行命令输入 docker search mysql 命令查看版本号;
4.用此命令语句 docker images 查看是否安装了 mysql;
5.用此命令语句 docker run -itd --name mysql-test -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql 运行 mysql 容器;
6.用此命令语句 docker ps 查看是否安装成功;
备注:接下来的计划是学习 SQL:如何创建数据表？数据表里面包含什么？如何在数据表里面插入数据？怎么查询数据？怎么更新数据？怎么删除数据？
