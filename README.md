## 如何将代码上传到 GitHub?

前置条件: Vscode代码软件安装好,必要的插件也安装好

第一步: 写一段 html 或 js 或 css 代码;

第二步: 在浏览是否能执行;

第三步: 先把要暂存的代码暂存, 用 git commit -m + ‘文件说明’ (用 git status -s 查看暂存内容);

第四步: 用 git push 拉取代码;

第五步: 用 git pull 同步代码;
